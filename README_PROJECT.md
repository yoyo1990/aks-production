AKS Production-Ready demo

This repository contains Terraform, container images and Kubernetes manifests for a small demo showing production-ready practices on AKS:
- IaC: Terraform for RG, ACR, AKS
- Apps: Service A (Python) and Service B (placeholder)
- Probes, Resource limits, NetworkPolicy, Ingress
